البري-بروسيسور (المعالج المسبق) هو برنامج يسمح بإستخدام وظائف CSS متوسعة تشمل إمكانية استخدام المتغيرات, الدوال, المشغلات البرمجية (معاملات) بالإضافة إلى العديد من الأصول التي تفتقدها CSS
